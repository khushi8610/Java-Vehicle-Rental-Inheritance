# Vehicle Rental System (Java)

This project is a simple Java program that demonstrates Object-Oriented Programming (OOP) concepts such as multilevel inheritance using a vehicle rental company example.

## Project Description
The program simulates a rental company that rents different types of vehicles. A base class `Vehicle` is created with basic vehicle information. The `Car` class extends `Vehicle`, and `ElectricCar` further extends `Car` to add electric vehicle features such as battery capacity and charging.

## Features
- Base class `Vehicle`
  - Variables: Brand, Model Year
  - Method: `start()`

- Derived class `Car`
  - Variable: Number of Doors
  - Method: `drive()`

- Derived class `ElectricCar`
  - Variable: Battery Capacity
  - Method: `charge()`
